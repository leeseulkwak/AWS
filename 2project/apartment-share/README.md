# 동네빌려 - 아파트 주민 물건 공유 서비스

## 서비스 개요

같은 아파트 단지 또는 인근 주민 간 물건을 단기간 대여할 수 있는 커뮤니티 기반 공유 서비스입니다.

### 핵심 가치

- **즉시 해결**: 급하게 필요한 물건을 바로 대여
- **비용 절감**: 구매 대신 대여로 경제적
- **커뮤니티 활성화**: 이웃 간 공유 문화 형성

### 주요 기능

1. **입주민 인증 시스템**: 검증된 주민들만 이용 가능
2. **비대면 거래**: 문고리 거래, 무인택배함, 관리사무소 이용
3. **사진 인증**: 수령/반납 시 물건 상태 사진 업로드
4. **보험 보장**: 삼성화재 보험으로 파손/분실 보상
5. **구매 연계**: 대여 후 구매 전환 마케팅

## 베타 테스트 계획

- **테스트 아파트**: 3~5개 대표 아파트
- **초기 물건**: 손상 리스크가 적은 물건 위주 (공구, 캠핑용품 등)
- **목표**: 데이터 수집 및 분석을 통한 사업 확장 판단

## 주요 KPI

- DAU/MAU (일일/월간 활성 사용자)
- 거래 성공률, 평균 대여 기간
- 재방문율, 이웃 간 거래 비율
- 리뷰 작성률, 평균 거래액
- 보험 청구율, 물건 파손율

## 기술 스택

- React 18
- TypeScript
- Vite
- React Router
- TailwindCSS
- Chart.js
- Lucide Icons

## 시작하기

```bash
# 패키지 설치
npm install

# 개발 서버 실행 (포트 3000)
npm run dev

# 빌드
npm run build

# 타입 체크
npm run typecheck
```

## 화면 구성

### 1. 홈 (/)
- 물건 탐색 및 검색
- 카테고리 필터링
- 서비스 통계 표시

### 2. 물건 상세 (/item/:id)
- 물건 정보 및 상태
- 대여 신청 폼
- 수령 방법 선택
- 보험 안내

### 3. 내 대여 (/my-rentals)
- 대여/대여중 물건 관리
- 사진 인증 업로드
- 거래 상태 추적

### 4. KPI 대시보드 (/kpi)
- 실시간 지표 시각화
- 카테고리별 분석
- 인사이트 및 제안

### 5. 마이페이지 (/profile)
- 사용자 정보 및 통계
- 뱃지 시스템
- 리뷰 관리

## 비즈니스 모델
